# Portfolio Website
This is a responsive portfolio website developed by the Love Kumar. It is built using HTML, CSS, and Bootstrap. The website showcases the skills, projects, and contact information of the portfolio owner.

# Features
- Responsive design that adapts to different screen sizes and devices.
- Home section with an introduction and background information.
- Projects section to showcase the portfolio owner's previous work.
- Contact section with a contact form and social media links.

# Technologies & Tools Used
- HTML
- CSS
- Bootstrap
- VS Code
